# 数据导入/导出 — 功能规格归档
> 从 PRD.md Section 3.5 归档。此功能已实现。

### 3.5 数据导入/导出

#### 3.5.1 导入

- **支持格式**：Excel（.xlsx）
- **兼容现有格式**：读取名为 "detail" 的工作表，固定列结构（日期、类型、金额、分组、描述、操作人）
- **导入模式**：全量覆盖（清空现有交易和操作人数据后重新导入）
- **导入流程**：
  1. 选择 Excel 文件
  2. 预览解析结果（展示有效记录数、操作人列表、分类列表、异常行提示）
  3. 用户确认后执行全量覆盖导入
- **分类处理**：自动匹配已有分类，未匹配的自动创建新分类
- **操作人处理**：清空后重建，根据导入数据自动创建
- **添加时间处理**：
  - 源文件包含「添加时间」列时，使用该时间作为 `created_at`，保持原始录入时间
  - 源文件不包含该列（旧格式，6列）时，以实际导入写入时间作为 `created_at`
  - 支持的时间格式：`YYYY-MM-DD HH:mm:ss`、Excel 日期时间序列号、纯日期（自动补全 `00:00:00`）

#### 3.5.2 导出

- **支持格式**：Excel（.xlsx）和 CSV（.csv），用户在导出时选择
- **导出列**：固定 7 列结构，包含添加时间：

| 列 | 内容 | 说明 |
|----|------|------|
| 日期 | `YYYY-MM-DD` | 交易日期 |
| 类型 | 消费 / 收入 / 投资 | 中文显示名 |
| 金额 | 数字（保留两位小数） | 负数原样导出（退款/冲正场景） |
| 分组 | 分类名称 | 列名为「分组」以兼容用户现有 Excel |
| 描述 | 自由文本 | 可为空 |
| 操作人 | 操作人名称 | 可为空 |
| **添加时间** | `YYYY-MM-DD HH:mm:ss` | **新增**，交易首次录入系统的时间 |

- **导出范围**：默认导出全部交易记录；支持按以下条件筛选后导出：
  - 日期范围（起止日期）
  - 交易类型（消费 / 收入 / 投资，可多选）
  - 分类（可多选）
  - 操作人（可多选）
  - 关键词（搜索描述字段）
- **排序**：导出数据按日期升序排列（与用户 Excel 习惯一致）
- **Excel 格式细节**：
  - 工作表名称为 `detail`（与导入格式一致，导出文件可直接作为导入源）
  - 首行为表头（日期、类型、金额、分组、描述、操作人、添加时间）
  - 添加时间列宽度 18 字符，时间格式按文本显示
- **CSV 格式细节**：
  - UTF-8 with BOM 编码（确保 Excel 打开时中文不乱码）
  - 首行为表头
  - 字段分隔符为逗号，含逗号或换行的字段用双引号包裹

##### 导出流程

1. 用户在设置页「数据管理」Tab 中点击「导出数据」按钮
2. 弹出导出配置面板，包含：
   - 格式选择：Excel / CSV（默认 Excel）
   - 筛选条件（可选，默认全部导出）：日期范围、类型、分类、操作人、关键词
   - 预览：显示符合条件的记录数
3. 点击「导出」按钮，弹出系统保存对话框
4. 默认文件名：`Moneta_导出_YYYYMMDD.xlsx`（或 `.csv`），日期为当天
5. 导出完成后提示：成功导出 N 条记录到 `文件路径`

##### 验收标准

| # | 验收条件 | 说明 |
|---|---------|------|
| AC-E1 | 支持导出为 Excel（.xlsx）格式 | 生成的文件可被 Excel / WPS 正常打开 |
| AC-E2 | 支持导出为 CSV（.csv）格式 | UTF-8 with BOM，Excel 打开中文不乱码 |
| AC-E3 | 导出列结构 | 日期、类型、金额、分组、描述、操作人、**添加时间** |
| AC-E4 | 导出的 Excel 可直接作为导入源 | 工作表名 `detail`，7列结构兼容（含添加时间） |
| AC-E16 | 导出包含添加时间列 | 格式 `YYYY-MM-DD HH:mm:ss`，24小时制 |
| AC-E17 | 添加时间保持原始值 | 导出→导入→导出循环，添加时间不变 |
| AC-E18 | 旧文件向后兼容 | 6列旧文件可正常导入，添加时间使用当前时间 |
| AC-E5 | 支持按日期范围筛选导出 | 选择起止日期后仅导出范围内记录 |
| AC-E6 | 支持按类型筛选导出 | 可选择一个或多个类型 |
| AC-E7 | 支持按分类筛选导出 | 可选择一个或多个分类 |
| AC-E8 | 支持按操作人筛选导出 | 可选择一个或多个操作人 |
| AC-E9 | 支持按关键词筛选导出 | 搜索描述字段 |
| AC-E10 | 默认导出全部记录 | 不设筛选条件时导出所有交易 |
| AC-E11 | 导出前显示符合条件的记录数 | 用户确认导出范围 |
| AC-E12 | 导出完成后提示成功信息 | 显示导出条数和文件路径 |
| AC-E13 | 无数据时给出提示 | 符合筛选条件的记录为 0 时提示，不生成空文件 |
| AC-E14 | 负数金额正确导出 | 退款/冲正的负数金额原样写入 |
| AC-E15 | 导出数据按日期升序排列 | 与用户 Excel 习惯一致 |

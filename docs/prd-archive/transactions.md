# 交易管理 — 功能规格归档
> 从 PRD.md Section 3.1 归档。此功能已实现。

### 3.1 交易管理

#### 3.1.1 交易数据结构

对齐用户现有 Excel 表结构：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 日期 | Date | ✅ | 默认为当前「记账日期」，可单条修改 |
| 添加时间 | DateTime | 系统生成 | 记录该条交易写入系统的时间，精确到秒，用于排序和审计，不可手动编辑 |
| 类型 | Enum | ✅ | "消费"、"收入" 或 "投资" |
| 金额 | Number | ✅ | 可为正数或负数，精确到分（两位小数）；负数用于退款、冲正等场景 |
| 分类 | String | ✅ | 从分类列表中选择 |
| 描述 | String | ❌ | 自由文本备注 |
| 操作人 | String | ❌ | 从已有操作人列表选择或输入新名称 |

#### 3.1.2 录入模式

1. **选定记账日期**：每次开始记账时，用户先选择一个日期（默认今天），本次会话中所有新增条目自动采用该日期
2. **快速录入表单**：选择类型 → 输入金额 → 选择分类 → （可选）填写描述和操作人 → 提交
3. **连续录入**：提交后表单自动清空（保留类型、日期、操作人），方便连续录入多条
4. **编辑与删除**：支持对已有交易的修改和删除

- 金额支持录入负数，用于退款、冲正等需要抵减原交易的场景

#### 3.1.3 交易列表

- 默认按日期倒序展示
- 支持按日期范围、类型、分类、操作人筛选
- 支持关键词搜索（搜索描述字段）
- 支持按添加时间排序，便于查看最新录入的数据
- 数据浏览页仅展示交易记录，分类和操作人管理已移至设置页（§3.3、§3.6）

#### 3.1.4 数据浏览与编辑

数据浏览页以单一表格形式展示所有交易记录（不含分类和操作人 Tab，这些在设置页管理）。

##### 表格结构

- 列：日期、添加时间、类型、金额、分类、描述、操作人、操作（编辑/删除按钮）
- 分页展示，支持切换每页条数（20/50/100）
- 表头显示总记录数
- 添加时间列为只读字段，显示系统实际写入该条记录的时间

##### 行内编辑

- 点击编辑按钮或双击行进入编辑状态，支持修改：
  - **日期**：日期选择器
  - **类型**：下拉选择（消费/收入/投资）
  - **金额**：数字输入框
  - **分类**：下拉选择（根据当前类型动态加载对应分类列表）
  - **描述**：文本输入框
  - **操作人**：下拉选择
- **添加时间**不可编辑，编辑已有记录时保持原值不变
- 类型变更后，若原分类不属于新类型，分类自动清空
- 编辑后保存即时写入数据库

##### 删除

- 每行操作列提供删除按钮
- 删除前弹出确认提示（防止误操作）
- 支持批量选择 + 批量删除

##### 新增一行

- 表格上方提供「新增」按钮
- 新增行各字段默认为空，但**日期字段记忆上次用户输入的值**（方便连续录入同一天的数据）
- 日期记忆仅当前会话有效，应用关闭后清空
- 新增行需填写必填字段（日期、类型、金额、分类）后方可保存

##### 按列排序

- 点击列头按该列升序/降序排序
- 支持排序的列：日期、添加时间、类型、金额、分类、操作人
- 添加时间排序基于该条记录的实际写入时间，而非记账日期
- 排序作用于全量数据结果，而非仅当前页前端重排

##### 日期范围筛选

数据浏览页工具栏支持日期范围筛选，帮助用户快速定位特定时间段的交易记录。

**筛选控件**
- **位置**：工具栏右侧，位于「搜索描述」输入框左侧
- **组件**：日期范围选择器（RangePicker）
- **格式**：YYYY-MM-DD，显示为「开始日期 ~ 结束日期」

**预设快捷选项**
点击日期选择器后下拉面板提供快捷选项：

| 选项 | 日期范围 |
|------|---------|
| 全部 | 清除日期筛选，显示全部数据 |
| 本月 | 当月 1 日 ~ 当月最后一天 |
| 上月 | 上月 1 日 ~ 上月最后一天 |
| 本季度 | 本季度首月 1 日 ~ 本季度末月最后一天 |
| 本年 | 当年 1 月 1 日 ~ 当年 12 月 31 日 |

**交互逻辑**
- 默认不限制日期范围，显示全部交易记录
- 选择日期范围后自动触发查询（防抖 300ms）
- 日期筛选与现有筛选条件（类型、分类、操作人、关键词）叠加生效
- 点击「全部」或清除按钮恢复显示全部数据
- 编辑/新增行时禁用日期筛选器（防止筛选条件变化导致编辑状态异常）

**边界处理**
- 开始日期 > 结束日期：自动交换两者顺序
- 选择的日期范围无数据：显示空状态提示「暂无符合条件的记录」

##### 从统计页返回

当用户从统计报表页通过「查看明细」功能跳转至数据浏览页后，提供快捷返回按钮，一键回到统计页并恢复之前的浏览状态。

**返回按钮显示逻辑**
- **显示条件**：当用户通过统计页趋势图的「查看明细」功能跳转过来时显示返回按钮
- **隐藏条件**：用户直接通过侧栏导航进入数据浏览页时不显示返回按钮
- **位置**：页面标题栏左侧，与「数据浏览」标题并排
- **样式**：返回箭头图标（←）+ 文字「返回统计页」，点击区域与标题保持适当间距

**状态恢复机制**
点击返回按钮后，应用导航回统计报表页（`/statistics`），并恢复以下状态：

| 状态类型 | 恢复内容 | 存储方式 |
|---------|---------|---------|
| **筛选条件** | 年份、类型（消费/收入/投资） | URL query parameters |
| **视图Tab** | 月度明细 / 年度汇总 | URL query parameters |
| **图表交互状态** | 当前独显的分类（如果有） | URL query parameters |

**URL 状态传递**
- 从统计页跳转至数据浏览页时，将当前统计页状态编码到 URL（如 `/?...&from=statistics&year=2024&type=expense&tab=monthly&soloCategory=正餐`）
- 返回时根据 `from=statistics` 判断显示返回按钮
- 返回按钮点击后，使用保存的状态重构统计页 URL（`/statistics?year=2024&type=expense&tab=monthly&soloCategory=正餐`）

**交互细节**
- 返回按钮使用浏览器历史 API 或 React Router 的 `navigate(-1)` 实现，确保返回流畅
- 若用户点击返回按钮前修改了数据浏览页的筛选条件，不影响返回功能（仅返回统计页，不携带数据浏览页的筛选状态）
- 返回后统计页的汇总卡片、交叉统计表、趋势图均基于恢复的状态重新加载数据

**边界处理**
- 用户复制带 `from=statistics` 的 URL 在新标签页打开：正常显示返回按钮，点击后正常跳转统计页
- 浏览器前进/后退操作：与返回按钮行为保持一致
- 从统计页跳转后手动修改 URL 删除 `from` 参数：返回按钮隐藏，视为直接访问数据浏览页

##### 验收标准

| # | 验收条件 | 说明 |
|---|---------|------|
| AC-D1 | 数据浏览页仅展示交易记录表格 | 不含分类和操作人 Tab |
| AC-D2 | 支持行内编辑所有字段 | 日期、类型、金额、分类、描述、操作人均可编辑 |
| AC-D3 | 类型变更后分类自动清空 | 切换类型后若分类不匹配则清空 |
| AC-D4 | 编辑即时保存到数据库 | 保存后刷新页面数据一致 |
| AC-D5 | 支持单行删除（有确认提示） | 防止误删 |
| AC-D6 | 支持批量选择和批量删除 | 勾选多行后一键删除 |
| AC-D7 | 支持新增一行 | 点击「新增」按钮添加空行 |
| AC-D8 | 新增行日期记忆上次输入值 | 连续添加时自动填充上次日期 |
| AC-D9 | 日期记忆会话内有效 | 应用关闭后清空 |
| AC-D10 | 必填字段校验 | 新增行未填日期/类型/金额/分类时不可保存 |
| AC-D11 | 支持按列排序 | 点击列头切换升序/降序 |
| AC-D12 | 表头显示总记录数 | 增删后实时更新 |
| AC-D13 | 每条交易自动记录添加时间 | 创建成功后可在数据浏览页看到该字段 |
| AC-D14 | 添加时间字段只读 | 新增和编辑时均不可手动修改 |
| AC-D15 | 支持按添加时间升序/降序排序 | 可查看最新录入或最早录入的数据 |
| AC-D16 | 添加时间排序基于真实写入时间 | 同一天的多条交易可按录入先后区分 |
| AC-D17 | 日期范围选择器显示在工具栏 | 位于搜索框左侧，布局协调 |
| AC-D18 | 支持选择开始日期和结束日期 | 使用 RangePicker 组件，格式 YYYY-MM-DD |
| AC-D19 | 预设快捷选项可用 | 本月、上月、本季度、本年、全部 |
| AC-D20 | 日期筛选与现有条件叠加 | 类型+分类+操作人+关键词+日期同时生效 |
| AC-D21 | 清除日期筛选后恢复全部 | 点击清除按钮或选择「全部」预设 |
| AC-D22 | 编辑/新增行时禁用筛选器 | 防止筛选条件变化导致编辑状态异常 |
| AC-D23 | 从统计页查看明细跳转后显示返回按钮 | 数据浏览页标题栏左侧显示「← 返回统计页」 |
| AC-D24 | 直接访问数据浏览页不显示返回按钮 | 通过侧栏导航进入时不显示返回按钮 |
| AC-D25 | 点击返回按钮回到统计页 | 使用 navigate(-1) 或类似方式返回 |
| AC-D26 | 返回后恢复统计页筛选条件 | 年份、类型筛选器恢复跳转前的值 |
| AC-D27 | 返回后恢复视图Tab状态 | 月度明细/年度汇总Tab恢复跳转前的选中状态 |
| AC-D28 | 返回后恢复图表独显状态 | 若跳转前有独显的分类，返回后保持独显 |
| AC-D29 | URL包含跳转来源标记 | 从统计页跳转时URL包含 `from=statistics` 参数 |
